# 🏠 PacadRent

PacadRent is a modern **Student Accommodation Platform** that helps users find and list PGs, rooms, and flats بسهولة (without brokers). It connects property owners directly with students and tenants.

---

## 🚀 Features

* 🔐 **Google Authentication** (Firebase)
* 🏡 **Add / Edit / Delete Property Listings**
* 📍 **Map Integration (Leaflet)**
* 💬 **Direct WhatsApp Contact with Owners**
* 🔍 **Advanced Search & Filters**

  * Price range
  * Property type
  * Rooms
  * Gender preference
  * Distance
  * Amenities
* ❤️ **Save/Favorite Listings**
* 📊 **Owner Dashboard**
* ⚡ **Real-time Database (Firebase Firestore)**
* 💳 **Premium Plan UI (Upgradeable)**

---

## 🛠️ Tech Stack

* **Frontend:** React (Vite)
* **Styling:** Tailwind CSS
* **Backend:** Firebase (Auth + Firestore)
* **Maps:** Leaflet / React-Leaflet
* **Deployment:** Vercel

---

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/pacadrent.git
cd pacadrent
```

### 2. Install dependencies

```bash
npm install
```

### 3. Create `.env` file

```env
VITE_FIREBASE_API_KEY=your_api_key
VITE_FIREBASE_AUTH_DOMAIN=your_auth_domain
VITE_FIREBASE_PROJECT_ID=your_project_id
```

### 4. Run locally

```bash
npm run dev
```

---

## 🌐 Deployment

This project is deployed using **Vercel**.

Steps:

1. Push code to GitHub
2. Import project in Vercel
3. Add environment variables
4. Click Deploy

## ⚖️ Legal

PacadRent acts as an **intermediary platform**.

* We do not own or verify listed properties
* Users are responsible for their listings and interactions
